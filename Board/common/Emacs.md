# Emacs

### Run Emacs in console:
emacs -nw

### Show Line number:
M-x linum mode

###  Run Shell Command:
M-! shell-command

### Kill Window:
C-x 0 Delete selected window
C-x 1 Delete other window

### Select other window:
C-x o

### Scroll other window:
C-M-v

### Switch Buffer:
C-x b (+ Up/Down arrow)

### undo:
C-/

### Copy:
M-w

### Cut
C-w

### Paste
C-y

### Paste next
M-y

### Delete Word
forward <del>    backward C-d
forward word M-<del> backward-word M-d
To end of line : C-k
To end of sentence : M-k
